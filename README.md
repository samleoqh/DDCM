### DDCM
This is a pytorch implementation of the DDCM-Net model as described in our paper: ["Dense Dilated Convolutions Merging Network for Semantic Segmentation"](https://arxiv.org/abs/1908.11799).	

### Key Requirements
- python==3.6.9
- pytorch==1.3.0
- opencv==3.4.2
- tensorboardx==1.9
- scikit-learn==0.21.3
